class pypes_client.cursesclient.MainWin(display)[source]¶

Bases: object

The main curses window.

Contains the in- and out streams that interact with the socket, and is responsible for handling the communication with the server. Actual display and user input is handled by the child components MainWin.input_field and MainWin.output_field.

Parameters

display (screen) – The main curses display. Dependency injected by the curses wrapper.
